# Changelog - BASIC GRAPH

## [1.0.0] - 2026-02-28

### Added
- Initial release of BASIC GRAPH with companion app BASIC GRAPH SP (Stripe Pay & Trial period)
- 6 circular data fields (1h, 3h, 5h, 7h, 9h, 11h positions)
- Graph field with Line and Bar display modes (up to 4 hours history)
- 3 classic top data fields (left, center, right)
- 4 info indicators
- Goal ring with configurable data type
- 73+ data types supported
- 22 customizable colors with 67 predefined palette
- 3 custom hexadecimal colors
- Dual weather support (Garmin Weather + OpenWeatherMap)
- 7 date display formats
- Multilingual support (19 languages)
- Intelligent power saving (screen saver + screen off)
